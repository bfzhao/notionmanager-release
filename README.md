## 🚀 Quick Start

**Windows**  
Double-click the `.exe` file and follow the installation wizard.

**macOS**  
1. Open the `.dmg` file  
2. Drag `NotionManager.app` into the `Applications` folder

**Linux**  
```bash
chmod +x NotionManager.AppImage
./NotionManager.AppImage
```
